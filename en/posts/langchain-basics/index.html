LangChain Basics - Building LLM Applications

Posted on 2026-05-27 Edited on 2026-07-20 In AI도구
The core building blocks of LangChain and how to build LLM applications, with examples.

What Is LangChain?

LangChain is an open-source framework that helps you quickly
build LLM-based applications. Beyond simple model calls, it provides
prompt management, external data integration, memory, tool
calling, and agent composition through a consistent interface.

LangChain’s core idea is the chain — connecting components like
a pipeline to build complex workflows.

Core Components

LangChain is made of these components.

Component Role
Model An LLM or ChatModel wrapper
Prompt A templated prompt
Output Parser Parses the response into a structured object
Retriever Searches relevant documents from a vector DB, etc.
Memory Manages conversation history
Tool An interface for calling external APIs/functions
Agent An LLM loop that selects and runs tools

Prompt Templates

Parameterize prompts for reusability.

LCEL (LangChain Expression Language)

LangChain’s modern interface is LCEL, which connects components with
the pipe operator (|). It has excellent readability, streaming,
and async support.

Streaming is simple too.

Building a RAG Pipeline

LangChain’s most popular use is Retrieval-Augmented Generation
(RAG). You embed external documents and store them in a vector DB,
fetch documents relevant to a question, and pass them to the LLM.

Agents and Tools

The pattern where the LLM selects and calls tools on its own is the
Agent.

An agent repeats this loop.

  1. The model decides whether to call a tool or answer directly
  2. On a tool call, it receives the result and passes it back to the model
  3. Repeat until the model produces a final answer

Memory Management

Conversational applications need to remember prior turns.

Best Practices

  • Separate prompts into external files: easier version control and A/B testing
  • Use output parsers: receive Pydantic models instead of free text
  • Adopt LangSmith: visually trace chain execution
  • Monitor cost: register token-usage callbacks
  • Set fallbacks: auto-switch to a backup model on failure

Wrapping Up

LangChain provides standardized building blocks for LLM
applications. Learn LCEL-based chain composition, RAG pipelines, and
the agent pattern, and you can quickly build practical AI systems
beyond a simple chatbot. That said, the abstraction is heavy — it’s
important to understand the internals as you use it.
