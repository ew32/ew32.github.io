Essential VSCode Shortcuts and Extensions

Posted on 2026-05-28 Edited on 2026-06-15 In DevEnvironment
Shortcuts and recommended extensions that maximize VSCode productivity, organized by category.

Why Use VSCode Fast

Visual Studio Code is a lightweight editor with a powerful
extension ecosystem. Having an environment where you can write,
navigate, and debug code without a mouse is the key to
productivity. This post introduces frequently used shortcuts and
extensions proven in practice.

Notation convention: Cmd on macOS maps to Ctrl on
Windows/Linux.

The Command Palette and Quick Navigation

Every VSCode command is accessible through the command palette.
If you just memorize the shortcuts, you can instantly invoke almost
every feature.

Feature macOS Windows/Linux
Command palette Cmd+Shift+P Ctrl+Shift+P
Quick open file Cmd+P Ctrl+P
Go to symbol Cmd+Shift+O Ctrl+Shift+O
Go to line Ctrl+G Ctrl+G
Go to definition F12 F12
Peek definition Option+F12 Alt+F12
Previous location Ctrl+- Alt+←

Editing Shortcuts

The core shortcuts that determine code-editing efficiency.

Feature macOS Windows/Linux
Duplicate line Shift+Option+↓ Shift+Alt+↓
Move line Option+↑/↓ Alt+↑/↓
Delete line Cmd+Shift+K Ctrl+Shift+K
Multi-cursor Option+click Alt+click
Add next match Cmd+D Ctrl+D
Select all matches Cmd+Shift+L Ctrl+Shift+L
Toggle comment Cmd+/ Ctrl+/
Format code Shift+Option+F Shift+Alt+F
Toggle word wrap Option+Z Alt+Z

Once you learn the multi-cursor (Cmd+D), the time spent on
simple repetitive work drops dramatically.

Search and Refactoring

Feature macOS Windows/Linux
Search in file Cmd+F Ctrl+F
Global search Cmd+Shift+F Ctrl+Shift+F
Global replace Cmd+Shift+H Ctrl+Shift+H
Rename F2 F2
Quick fix Cmd+. Ctrl+.

Rename with F2 isn’t a simple text replacement — it traces all
LSP-based symbol references, so it’s safe.

Split Screen and Terminal

Feature macOS Windows/Linux
Split editor Cmd+\ Ctrl+\
Move to group Cmd+1/2/3 Ctrl+1/2/3
Toggle sidebar Cmd+B Ctrl+B
Toggle terminal Ctrl+` Ctrl+`
Split terminal Cmd+\ (in terminal) Ctrl+Shift+5

Recommended Extensions

Universal Essentials

  • Prettier: a code formatter (JS/TS/CSS/JSON, etc.)
  • ESLint: a JavaScript/TypeScript linter
  • GitLens: inline Git info, blame, history
  • Error Lens: shows errors inline next to the line
  • Path Intellisense: file path auto-completion

By Language

Language Recommended extensions
Python Python, Pylance, Black Formatter
JavaScript/TS ESLint, Prettier, TypeScript Hero
Go Go (official)
Rust rust-analyzer
Java Extension Pack for Java
Docker Docker, Dev Containers

Collaboration and Productivity

  • GitHub Copilot: AI auto-completion
  • Live Share: real-time collaboration
  • REST Client: manage HTTP requests in .http files
  • Todo Tree: collect TODO, FIXME comments in one view
  • Project Manager: quickly switch between multiple projects

Recommended settings.json

You can fine-tune settings by editing settings.json directly.

Using Workspaces

Putting project-specific settings in .vscode/settings.json lets
the whole team share them. Specifying recommended extensions in
.vscode/extensions.json gives new team members an auto-install
prompt.

Wrapping Up

The key to VSCode productivity is a shortcut habit that reduces
mouse use and choosing extensions that fit your purpose.
Getting even 10 of the shortcuts in this post into your hands will
noticeably speed up your coding. Rather than memorizing them all at
once, consciously use two or three each week.
