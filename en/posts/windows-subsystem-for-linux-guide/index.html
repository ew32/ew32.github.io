A Complete WSL2 Guide: Using Linux on Windows

Posted on 2025-12-17 Edited on 2026-05-22 In DevEnvironment , System
An introduction to the Linux development environment for Windows, covering the concept, advantages, installation, and usage points of WSL2.

What Is WSL?

WSL (Windows Subsystem for Linux) is a compatibility layer that
lets you run Linux binary executables natively on Windows 10 and
Windows 11. It lets you use a Linux environment on Windows without
a virtual machine or dual boot, making it a very useful tool for
developers.

WSL1 vs WSL2

WSL1

  • Translates system calls to the Windows kernel
  • Fast file system performance (when accessing Windows files)
  • Lower memory usage

WSL2 (recommended)

  • A lightweight virtual machine using a real Linux kernel
  • Full system call compatibility
  • Better performance (especially for I/O work)
  • Perfect Linux compatibility
  • Fast speed

In most cases, WSL2 is recommended.

Advantages of WSL

1. Integrated Dev Environment

You can use Windows GUI applications and Linux command-line tools
together.

2. Resource Efficiency

It uses far fewer resources than a virtual machine and has almost
no boot time.

3. File System Access

Mutual access between the Windows and Linux file systems is
possible.

4. IDE Integration

Major IDEs such as Visual Studio Code and IntelliJ support WSL
directly.

5. A Full Linux Experience

You can use package managers (apt, yum, etc.) and all Linux tools
as-is.

How to Install WSL2

Prerequisites

  • Windows 10 version 2004 or later (build 19041 or later), or Windows 11
  • A 64-bit processor
  • Virtualization enabled (set in the BIOS)

Method 1: Simple Install (Windows 11 & recent Windows 10)

1. Run PowerShell as administrator

2. Run the WSL install command

This command automatically:

  • Enables the WSL feature
  • Enables the Virtual Machine Platform
  • Updates the Linux kernel
  • Sets WSL2 as the default version
  • Installs the Ubuntu distribution

3. Reboot the system

4. Initial Ubuntu setup
After rebooting, Ubuntu runs automatically and you set a username
and password.

Wrapping Up

WSL2 is the most efficient way to build a Linux development
environment on Windows. It provides both the full isolation of a
virtual machine and near-native performance, and it’s perfectly
compatible with most Linux tools.

Since you can leverage the convenience of Windows and the power of
Linux at the same time, it’s ideal for cross-platform development
or testing server environments. Once set up, you can move smoothly
between Windows and Linux, greatly improving productivity.

In particular, the integration with Visual Studio Code, the
excellent UX of Windows Terminal, and the two-way file system
accessibility make WSL2 an essential tool for developers.
