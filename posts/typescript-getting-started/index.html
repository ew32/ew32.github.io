TypeScript란?

TypeScript는 Microsoft에서 개발한 JavaScript의 슈퍼셋(superset) 언어로, JavaScript에 정적 타입 시스템을 추가한 언어입니다. 컴파일 시점에 타입을 검사하여 런타임 오류를 줄여주고, IDE의 자동 완성과 리팩토링을 강력하게 지원합니다.

왜 TypeScript인가?

  • 타입 안정성: 컴파일 시점에 오류 발견
  • 개발 생산성: IDE 자동 완성, 인라인 문서
  • 유지보수성: 큰 코드베이스에서 변경 영향 추적 용이
  • 점진적 도입: JavaScript 코드를 그대로 사용 가능
