мһ‘м„ұмқј 2026-05-11 Edited on 2026-07-20 In Backend , Database
м»Өл„Ҙм…ҳ н’Җмқҙ н•„мҡ”н•ң мқҙмң мҷҖ HikariCPмқҳ лҸҷмһ‘ мӣҗлҰ¬, н•өмӢ¬ м„Өм • л°Ҹ нҠңлӢқ нҸ¬мқёнҠёлҘј мӮҙнҺҙлҙ…лӢҲлӢӨ.

м»Өл„Ҙм…ҳмқҖ мҷң л№„мӢјк°Җ

лҚ°мқҙн„°лІ мқҙмҠӨ м—°кІ°мқ„ мғҲлЎң л§Ңл“Ө л•Ңл§ҲлӢӨ лӢӨмқҢ мһ‘м—…мқҙ мқјм–ҙлӮ©лӢҲлӢӨ.

  1. TCP 3-way handshake
  2. (TLS мӮ¬мҡ© мӢң) TLS н•ёл“ңм…°мқҙнҒ¬
  3. мқёмҰқ, к¶Ңн•ң нҷ•мқё
  4. м„ём…ҳ ліҖмҲҳ мҙҲкё°нҷ”
  5. л©”лӘЁлҰ¬ н• лӢ№

л§Ө мҡ”мІӯл§ҲлӢӨ мқҙ кіјм •мқ„ кұ°м№ҳл©ҙ мқ‘лӢө мӢңк°„мқҙ мҲҳмӢӯ ms мқҙмғҒ лҠҳм–ҙлӮҳкі , DB м„ңлІ„ мёЎм—җлҸ„ л¶Җн•ҳк°Җ нҒҪлӢҲлӢӨ. м»Өл„Ҙм…ҳ н’Җ(Connection Pool) мқҖ лҜёлҰ¬ л§Ңл“Өм–ҙл‘” м»Өл„Ҙм…ҳмқ„ мһ¬мӮ¬мҡ©н•ҙ мқҙ л№„мҡ©мқ„ м ңкұ°н•©лӢҲлӢӨ.

м»Өл„Ҙм…ҳ н’Җмқҳ лҸҷмһ‘

close()лҠ” мӢӨм ңлЎң м»Өл„Ҙм…ҳмқ„ лҒҠлҠ” кІғмқҙ м•„лӢҲлқј н’Җм—җ л°ҳнҷҳн•©лӢҲлӢӨ. л”°лқјм„ң try-with-resourcesлӮҳ finallyм—җм„ң л°ҳл“ңмӢң л°ҳнҷҳн•ҙм•ј н•©лӢҲлӢӨ.

HikariCPмқҳ нҠ№м§•

HikariCPлҠ” Spring Boot 2.xл¶Җн„° кё°ліё м»Өл„Ҙм…ҳ н’Җмһ…лӢҲлӢӨ.

  • л§Өмҡ° к°ҖлІјмҡҙ мҪ”л“ң лІ мқҙмҠӨ
  • ConcurrentBag мһҗлЈҢкө¬мЎ°лЎң лқҪ кІҪн•© мөңмҶҢнҷ”
  • л№ лҘё к°Җм ёмҳӨкё°В·л°ҳнҷҳ кІҪлЎң
  • м •нҷ•н•ң л©”нҠёлҰӯ(HikariPool.MBean)

н•өмӢ¬ м„Өм •

мҶҚм„ұ мқҳлҜё к¶ҢмһҘ кё°ліёк°’
maximumPoolSize мөңлҢҖ м»Өл„Ҙм…ҳ мҲҳ CPU/мӣҢнҒ¬лЎңл“ңлі„ мӮ°м •
minimumIdle мөңмҶҢ мң нңҙ м»Өл„Ҙм…ҳ maximumPoolSizeмҷҖ лҸҷмқј(к¶ҢмһҘ)
connectionTimeout н’Җм—җм„ң м»Өл„Ҙм…ҳмқ„ кё°лӢӨлҰ¬лҠ” мөңлҢҖ мӢңк°„ 30,000 ms
idleTimeout мң нңҙ м»Өл„Ҙм…ҳ мң м§Җ мӢңк°„ 600,000 ms
maxLifetime м»Өл„Ҙм…ҳ мөңлҢҖ мҲҳлӘ… 1,800,000 ms
validationTimeout м»Өл„Ҙм…ҳ мң нҡЁм„ұ кІҖмӮ¬ мӢңк°„ 5,000 ms
leakDetectionThreshold лҲ„мҲҳ к°җм§Җ мӢңк°„ мҡҙмҳҒм—җм„  5,000 ms м •лҸ„

Spring Boot мҳҲмӢң:

н’Җ нҒ¬кё° мӮ°м •

HikariCP мң„нӮӨмқҳ кіөмӢқмқҙ к°ҖмһҘ мһҗмЈј мқёмҡ©лҗ©лӢҲлӢӨ.

лҢҖл¶Җл¶„мқҳ OLTP мӣҢнҒ¬лЎңл“ңм—җм„ңлҠ” 10~30 м •лҸ„л©ҙ м¶©л¶„н•©лӢҲлӢӨ. н’Җмқҙ нҒҙмҲҳлЎқ мўӢлӢӨлҠ” кІғмқҖ нқ”н•ң мҳӨн•ҙмқҙл©°, DB мёЎ м»Ён…ҚмҠӨнҠё мҠӨмң„м№ӯкіј лқҪ кІҪн•©мңјлЎң мҳӨнһҲл Ө м„ұлҠҘмқҙ л–Ём–ҙм§‘лӢҲлӢӨ.

мһ‘мқҖ н’Җмқҙ лҚ” л№ лҘё мқҙмң 

  • DBлҠ” лҸҷмӢңм—җ мІҳлҰ¬н•  мҲҳ мһҲлҠ” мһ‘м—… мҲҳк°Җ м ңн•ңм Ғ(CPUВ·л””мҠӨнҒ¬)
  • н’Җмқҙ нҒ¬л©ҙ DBм—җ лҢҖкё° нҒҗк°Җ мҢ“м—¬ мқ‘лӢө мӢңк°„мқҙ мҳӨнһҲл Ө мҰқк°Җ
  • мһ‘мқҖ н’Җ + л№ лҘё нҡҢм „мқҙ лҚ” лҶ’мқҖ мІҳлҰ¬лҹүмқ„ л§Ңл“ӯлӢҲлӢӨ

maxLifetimeкіј DB нғҖмһ„м•„мӣғ м •л ¬

DBмқҳ wait_timeout(MySQL) лҳҗлҠ” л°©нҷ”лІҪмқҳ idle TCP нғҖмһ„м•„мӣғліҙлӢӨ maxLifetimeмқҙ м§§м•„м•ј н•©лӢҲлӢӨ. к·ёл Үм§Җ м•Ҡмңјл©ҙ лҒҠм–ҙм§„ мўҖл№„ м»Өл„Ҙм…ҳмқ„ н’Җмқҙ лӘЁлҘҙкі  лӮҙмЈјкІҢ лҗ©лӢҲлӢӨ.

лҲ„мҲҳ к°җм§Җ

leakDetectionThresholdлҘј м„Өм •н•ҳл©ҙ м •н•ҙм§„ мӢңк°„ лҸҷм•Ҳ л°ҳнҷҳлҗҳм§Җ м•ҠмқҖ м»Өл„Ҙм…ҳмқ„ лЎңк·ёлЎң м•Ңл ӨмӨҚлӢҲлӢӨ.

мҠӨнғқ нҠёл ҲмқҙмҠӨлҘј нҶөн•ҙ close() лҲ„лқҪ м§Җм җмқ„ м°ҫм•„лӮј мҲҳ мһҲмҠөлӢҲлӢӨ.

л©”нҠёлҰӯ лӘЁлӢҲн„°л§Ғ

HikariCPлҠ” MicrometerмҷҖ нҶөн•©лҗҳм–ҙ лӢӨмқҢ м§Җн‘ңлҘј л…ём¶ңн•©лӢҲлӢӨ.

  • hikaricp.connections.active
  • hikaricp.connections.idle
  • hikaricp.connections.pending (лҢҖкё° мӨ‘ мҠӨл Ҳл“ң)
  • hikaricp.connections.usage (мӮ¬мҡ© мӢңк°„ л¶„нҸ¬)
  • hikaricp.connections.acquire (к°Җм ёмҳӨлҠ” лҚ° кұёлҰ° мӢңк°„)

pendingмқҙ 0ліҙлӢӨ мһҗмЈј нҒ¬лӢӨл©ҙ н’Җ нҒ¬кё° л¶ҖмЎұ, acquire мӢңк°„мқҙ кёёлӢӨл©ҙ DB мқ‘лӢө м§Җм—°мқ„ мқҳмӢ¬н•©лӢҲлӢӨ.

нқ”н•ң н•Ём •

  • нҠёлһңмһӯм…ҳмқ„ л§Өмҡ° кёёкІҢ мһЎмқҢ: мҷёл¶Җ API нҳём¶ңмқ„ нҠёлһңмһӯм…ҳ м•Ҳм—җ л‘җм§Җ л§Ҳм„ёмҡ”.
  • @Transactionalмқҙ л¶ҷмқҖ л©”м„ңл“ңм—җм„ң лҸҷкё° мҷёл¶Җ нҳём¶ң: н’Җмқҙ л№ лҘҙкІҢ кі к°Ҳлҗ©лӢҲлӢӨ.
  • N+1 мҝјлҰ¬: м»Өл„Ҙм…ҳ мӮ¬мҡ© мӢңк°„мқ„ мҰқк°ҖмӢңнӮөлӢҲлӢӨ.
  • мһҘкё° мӢӨн–ү л°°м№ҳмҷҖ OLTPк°Җ к°ҷмқҖ н’Җмқ„ кіөмң : лі„лҸ„ лҚ°мқҙн„°мҶҢмҠӨлЎң л¶„лҰ¬.
  • PreparedStatement мәҗмӢң лҜёмӮ¬мҡ©: cachePrepStmts=trueлҘј мјңл‘җм„ёмҡ”.

лӢӨмӨ‘ лҚ°мқҙн„°мҶҢмҠӨ

мқҪкё° ліөм ңліёмқ„ нҷңмҡ©н•  л•Ң н’Җмқ„ л¶„лҰ¬н•©лӢҲлӢӨ.

@Transactional(readOnly = true)мҷҖ лқјмҡ°нҢ… лҚ°мқҙн„°мҶҢмҠӨлҘј мЎ°н•©н•ҳл©ҙ л¶Җн•ҳ л¶„мӮ°мқҙ к°ҖлҠҘн•©лӢҲлӢӨ.

нҠңлӢқ м Ҳм°Ё мҡ”м•Ҫ

  1. мІҳмқҢм—” мһ‘кІҢ мӢңмһ‘ (10~20)
  2. л¶Җн•ҳ н…ҢмҠӨнҠёлЎң pending, acquire кҙҖм°°
  3. DB CPU/IOлҘј н•Ёк»ҳ лӘЁлӢҲн„°л§Ғ
  4. DBк°Җ н•ңкі„лқјл©ҙ н’Җмқ„ лҠҳлҰ¬м§Җ л§җкі  DBВ·мҝјлҰ¬л¶Җн„° мөңм Ғнҷ”
  5. maxLifetimeмқ„ DBВ·л„ӨнҠёмӣҢнҒ¬ нғҖмһ„м•„мӣғ лҜёл§ҢмңјлЎң м„Өм •

л§Ҳл¬ҙлҰ¬

м»Өл„Ҙм…ҳ н’ҖмқҖ лӢЁмҲңн•ң м„Өм • к°ҷм§Җл§Ң мӢңмҠӨн…ңмқҳ мқ‘лӢө мӢңк°„кіј м•Ҳм •м„ұм—җ м§Ғм ‘ мҳҒн–Ҙмқ„ мӨҚлӢҲлӢӨ. HikariCPмқҳ кё°ліёк°’мқҖ н•©лҰ¬м Ғмқҙм§Җл§Ң, н’Җ нҒ¬кё°мҷҖ мҲҳлӘ… м„Өм •мқҖ нҷҳкІҪм—җ л§һкІҢ мЎ°м •н•ҙм•ј н•©лӢҲлӢӨ. вҖңн’Җмқ„ нҒ¬кІҢ мһЎмңјл©ҙ л№Ёлқјм§„лӢӨвҖқлҠ” м§ҒкҙҖ лҢҖмӢ , DBк°Җ мӢӨм ңлЎң мҶҢнҷ”н•  мҲҳ мһҲлҠ” л§ҢнҒјл§Ң ліҙлӮҙкі  лӮҳлЁём§ҖлҠ” нҒҗлЎң нқҳлҹ¬к°ҖкІҢ л‘җлҠ” кІғмқҙ к°ҖмһҘ мўӢмқҖ нҠңлӢқмһ…лӢҲлӢӨ.
