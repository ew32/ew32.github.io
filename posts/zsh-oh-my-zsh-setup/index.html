Zsh와 Oh My Zsh로 셸 환경 꾸미기

작성일 2026-07-16 Edited on 2026-05-22 In 개발환경
Zsh의 강점과 Oh My Zsh, Powerlevel10k, 자동완성/하이라이트 플러그인으로 생산성 높은 터미널을 구성하는 방법.

Zsh를 쓰는 이유

Zsh는 Bash와 호환되면서 더 강력한 자동완성, 풍부한 글로브, 테마 시스템을 제공합니다. macOS의 기본 셸이 Zsh로 바뀐 이후 사실상 표준이 되었습니다. Oh My Zsh는 그 위에 플러그인/테마 매니저를 얹어 설정 부담을 크게 줄여줍니다.

설치

설치하면 홈 디렉터리에 ~/.zshrc가 생성됩니다.

Powerlevel10k 테마

가볍고 정보가 풍부한 프롬프트 테마입니다. git 상태, 가상환경, 종료 코드, 실행 시간 등을 한눈에 보여줍니다.

~/.zshrc의 테마를 변경합니다.

새 터미널을 열면 p10k configure 마법사가 시작됩니다.

필수 플러그인

세 개만 설치해도 체감 차이가 큽니다.

~/.zshrc의 plugins 라인을 수정합니다.

  • git: gst, gco, gp 등 수십 개의 git 별칭
  • z: 자주 가던 디렉터리로 점프 (z proj)
  • zsh-autosuggestions: 히스토리 기반 회색 제안, →로 수락
  • zsh-syntax-highlighting: 명령어 색상 강조

별칭과 함수

~/.zshrc 끝에 자주 쓰는 명령을 별칭으로 등록합니다.

히스토리 강화

기본 히스토리는 너무 작고 휘발성이 큽니다.

fzf 결합

퍼지 검색기 fzf를 더하면 히스토리(Ctrl+R), 파일 검색이 매우 빨라집니다.

성능 점검

플러그인이 늘수록 셸 시작이 느려질 수 있습니다.

500ms를 넘어가면 zsh-defer로 지연 로딩하거나 불필요한 플러그인을 제거합니다. Powerlevel10k는 instant prompt 기능으로 체감 지연을 크게 줄여줍니다.

마무리

Zsh + Oh My Zsh + Powerlevel10k + 자동완성/하이라이트 조합은 5분 투자로 평생의 생산성을 얻는 설정입니다. 자신의 워크플로에 맞게 별칭과 플러그인을 조금씩 늘려가며 .zshrc를 다듬어 보세요.
