작성일 2026-05-19 Edited on 2026-05-20 In Frontend , JavaScript

구조 분해 할당이란?

구조 분해 할당(Destructuring Assignment) 은 배열이나 객체의 값을 추출하여 별도의 변수에 한 번에 할당하는 ES6 문법입니다. 코드를 훨씬 간결하고 명확하게 만들어주며, 함수 매개변수, 반환값 처리, 상태 업데이트 등에 폭넓게 활용됩니다.
