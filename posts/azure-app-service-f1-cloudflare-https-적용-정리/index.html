Azure App Service F1 + Cloudflare HTTPS 적용 정리

개요

Azure App Service F1(Free) 요금제는:

  • 사용자 지정 도메인(Custom Domain) 미지원
  • 사용자 지정 도메인 HTTPS 미지원

따라서 Cloudflare를 이용해 HTTPS 프록시 구성 필요.

구조:

사용자
→ Cloudflare HTTPS
→ Cloudflare Worker
→ Azure App Service(F1)

1. Azure App Service 확인

Azure 기본 주소 확인:

https://.azurewebsites.net

이 주소는 정상 접속되어야 함.

4. Cloudflare 네임서버 적용

Cloudflare가 네임서버 2개 제공.

예시:

.ns.cloudflare.com .ns.cloudflare.com

가비아 → 도메인 관리 → 네임서버 설정

기존 네임서버 제거 후
Cloudflare 네임서버 입력.

DNS 전파:

  • 보통 5~30분
  • 길면 1~2시간

12. 최종 결과

가능:

  • F1 무료 유지
  • 사용자 지정 도메인 사용
  • HTTPS 적용
  • Cloudflare CDN 사용 가능

주의:

  • Worker 무료 호출 제한 존재
  • 일부 웹소켓/스트리밍은 추가 설정 필요 가능
  • 앱 내부 hostname 인식 주의
