VSCode 필수 단축키와 확장 프로그램

작성일 2026-05-28 Edited on 2026-07-20 In 개발환경
VSCode 생산성을 극대화하는 단축키와 추천 확장 프로그램을 카테고리별로 정리합니다.

VSCode를 빠르게 쓰는 이유

Visual Studio Code는 가벼우면서도 강력한 확장 생태계를 가진 에디터입니다. 마우스 없이도 코드 작성, 탐색, 디버깅이 가능한 환경을 갖추는 것이 생산성의 핵심입니다. 이 글에서는 자주 쓰는 단축키와 실무에서 검증된 확장 프로그램을 소개합니다.

표기 규칙: macOS는 Cmd, **Windows/Linux는 Ctrl**로 매핑됩니다.

명령 팔레트와 빠른 이동

VSCode의 모든 명령은 명령 팔레트로 접근할 수 있습니다. 단축키만 외워두면 거의 모든 기능을 즉시 호출 가능합니다.

기능 macOS Windows/Linux
명령 팔레트 Cmd+Shift+P Ctrl+Shift+P
파일 빠른 열기 Cmd+P Ctrl+P
심볼로 이동 Cmd+Shift+O Ctrl+Shift+O
라인으로 이동 Ctrl+G Ctrl+G
정의로 이동 F12 F12
정의 미리보기 Option+F12 Alt+F12
이전 위치 Ctrl+- Alt+←

편집 단축키

코드 편집 효율을 결정하는 핵심 단축키들입니다.

기능 macOS Windows/Linux
라인 복제 Shift+Option+↓ Shift+Alt+↓
라인 이동 Option+↑/↓ Alt+↑/↓
라인 삭제 Cmd+Shift+K Ctrl+Shift+K
멀티 커서 Option+클릭 Alt+클릭
다음 일치 추가 Cmd+D Ctrl+D
모두 일치 선택 Cmd+Shift+L Ctrl+Shift+L
주석 토글 Cmd+/ Ctrl+/
코드 포맷팅 Shift+Option+F Shift+Alt+F
줄바꿈 토글 Option+Z Alt+Z

멀티 커서(Cmd+D)는 한 번 익히면 단순 반복 작업 시간이 극적으로 줄어듭니다.

검색과 리팩터링

기능 macOS Windows/Linux
파일 내 검색 Cmd+F Ctrl+F
전역 검색 Cmd+Shift+F Ctrl+Shift+F
전역 치환 Cmd+Shift+H Ctrl+Shift+H
이름 변경 F2 F2
빠른 수정 Cmd+. Ctrl+.

F2 이름 변경은 단순 텍스트 치환이 아니라 LSP 기반 심볼 참조를 모두 추적하므로 안전합니다.

분할 화면과 터미널

기능 macOS Windows/Linux
에디터 분할 Cmd+\ Ctrl+\
그룹 이동 Cmd+1/2/3 Ctrl+1/2/3
사이드바 토글 Cmd+B Ctrl+B
터미널 토글 Ctrl+` Ctrl+`
터미널 분할 Cmd+\ (터미널) Ctrl+Shift+5

추천 확장 프로그램

공통 필수

  • Prettier: 코드 포맷터 (JS/TS/CSS/JSON 등)
  • ESLint: JavaScript/TypeScript 린터
  • GitLens: Git 정보 인라인 표시, blame, history
  • Error Lens: 오류를 라인 옆에 인라인으로 표시
  • Path Intellisense: 파일 경로 자동완성

언어별

언어 추천 확장
Python Python, Pylance, Black Formatter
JavaScript/TS ESLint, Prettier, TypeScript Hero
Go Go (공식)
Rust rust-analyzer
Java Extension Pack for Java
Docker Docker, Dev Containers

협업 및 생산성

  • GitHub Copilot: AI 자동완성
  • Live Share: 실시간 협업
  • REST Client: HTTP 요청을 .http 파일로 관리
  • Todo Tree: TODO, FIXME 주석 모아보기
  • Project Manager: 여러 프로젝트 간 빠른 전환

settings.json 추천 설정

설정은 settings.json에서 직접 편집하면 정밀하게 조정할 수 있습니다.

워크스페이스 활용

프로젝트별 설정은 .vscode/settings.json에 두면 팀 전체가 공유할 수 있습니다. .vscode/extensions.json에 추천 확장을 명시해두면 새 팀원이 자동 설치를 안내받습니다.

마무리

VSCode 생산성의 핵심은 마우스 사용을 줄이는 단축키 습관과 목적에 맞는 확장 선택입니다. 이 글에 정리된 단축키 중 10개만 손에 익혀도 코드 작성 속도가 눈에 띄게 빨라집니다. 한 번에 다 외우려 하기보다 매주 두세 개씩 의식적으로 사용해 보세요.
