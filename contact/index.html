문의하기

블로그 글에 대한 질문, 오류 제보, 주제 제안 등은 아래 채널로 연락 주세요.

이메일

  • 문의 이메일: [YOUR_EMAIL]

일반적으로 영업일 기준 3~5일 이내에 답변드립니다. 주말·공휴일에는 응답이 지연될 수 있습니다.

GitHub

소스 코드, 코드 관련 이슈, 풀 리퀘스트는 GitHub에서 다뤄주세요.

  • GitHub: https://github.com/ew32

문의 시 안내

원활한 답변을 위해 다음 내용을 함께 보내주시면 도움이 됩니다.

  1. 글 URL 또는 제목 — 어떤 글에 대한 문의인지 명확히 알 수 있도록
  2. 환경 정보 — OS, 언어/프레임워크 버전 등 (코드 관련 질문인 경우)
  3. 재현 절차 — 오류 재현 단계 또는 기대 결과 vs 실제 결과
  4. 연락 가능한 이메일 — 답신 가능한 주소

답변하기 어려운 문의

다음 유형의 문의에는 답변이 어려울 수 있습니다.

  • 개인 코드 리뷰·디버깅 요청
  • 학교/회사 과제 대행 요청
  • 광고/홍보 목적 메일
